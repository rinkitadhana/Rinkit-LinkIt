## Getting Started

<!-- Run Locally -->
### Make your LinkIt

Clone the project

```bash
  git clone git@github.com:rinkitadhana/LinkIt.git
```

Go to the project directory

```bash
  cd my-project
```

npm Install

```bash
  npm i
```

Run the vite app

```bash
  npm run dev
```
